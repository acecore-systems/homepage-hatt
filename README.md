# 概要

Hattのホームページのマスターソースです。

絵・小説・VRChat向け3Dアバター/ギミック制作をまとめる、Astro製の静的サイトです。

# 技術

| 使用箇所       | 使用技術           |
| -------------- | ------------------ |
| 言語           | TypeScript, Astro  |
| フレームワーク | Astro              |
| CSS            | UnoCSS, custom CSS |
| CMS            | Sveltia CMS        |
| 検索           | Pagefind           |

# 開発

```bash
npm install
npm run dev
```

# ビルド

```bash
npm run build
```
